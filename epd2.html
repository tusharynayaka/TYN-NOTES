Regulated Power Supply

Power supplies are designed for a dedicated load and environment. But sometimes main supply voltage, load, and surrounding temperature keep changing, altering the component parameters and hence changing the output voltage.

If the power supply output voltage changes, chances are it may cross these current thresholds, which are undesirable for electronic devices that work within power limits.

Regulated power supply converts an alternating current signal to a constant signal.

Fig 1: Block diagram of regulated power supply
120 V rms → Transformer → Rectifier → Filter → IC Regulator → Load
(Each stage progressively smooths the signal: sinusoidal AC → half-wave/full-wave DC → ripple-reduced DC → constant regulated DC)

Block Diagram Components

  • Transformer: The output sinusoidal voltage is either step-up or step-down from the input sinusoidal voltage value.
  • Rectifier: Converts an alternating current into a direct one by allowing the current to flow through it in one direction only.
  • Filter: Removes the AC ripples from the DC signal obtained from the rectifier.
  • Regulator: Converts DC voltage into a lower constant voltage.

Half Wave Rectifier (HWR)

HWR – Ideal Diode

The input waveform is \(v_i = V_m \sin \omega t\). The HWR circuit consists of a diode in series with a load resistor \(R_L\).

Positive Half Cycle:

Conduction region \(0 \to T/2\). The diode is forward biased.

\[ I_m = \frac{V_m}{R_L}, \quad V_o = V_m \]

Negative Half Cycle:

Non-conduction region \(T/2 \to T\). The diode is reverse biased.

\[ I = 0, \quad V_o = 0 \]
Input Waveform: Sinusoidal signal \(v_i = V_m \sin\omega t\). Peak at \(V_m\), negative trough at \(-V_m\).
Output Waveform: Only the positive half-cycles appear; output is zero during negative half. \(V_{dc} = 0.318\,V_m\).

HWR – Non-Ideal Diode

For a silicon diode with knee voltage \(V_k = 0.7\,\text{V}\), the diode only conducts when \(v_i > V_k\). The output is offset by \(V_k\).

\[ V_{o\,\max} = V_m - V_k \] \[ I_{o\,\max} = \frac{V_m - V_k}{R_L} \]
Effect of \(V_k\) on half-wave rectified signal: The output is shifted downward by \(V_k\). The conduction starts slightly later in the positive cycle and the peak output is reduced by the diode drop.

Average Voltage / Current for HWR – Ideal Diode

\[ V_{dc} = \frac{1}{2\pi} \int_0^{2\pi} V_m \sin\omega t\, d\omega t = \frac{1}{2\pi} \int_0^{\pi} V_m \sin\omega t\, d\omega t = \frac{V_m}{2\pi} \left[-\cos\omega t\right]_0^{\pi} = \frac{V_m}{\pi} \]
\[ \boxed{V_{dc} = \frac{V_m}{\pi} = 0.318\,V_m \quad (\text{half-wave})} \] \[ I_{dc} = \frac{V_{dc}}{R_L} = \frac{V_m}{\pi R_L} = \frac{I_m}{\pi} \]

RMS Voltage / Current for HWR – Ideal Diode

\[ V_{rms} = \left[\frac{1}{2\pi}\int_0^{2\pi}(V_m\sin\omega t)^2\,d\omega t\right]^{1/2} = \left[\frac{V_m^2}{2\pi}\int_0^{\pi}\sin^2\omega t\,d\omega t\right]^{1/2} = \left[\frac{V_m^2}{2\pi}\int_0^{\pi}\frac{1-\cos 2\omega t}{2}\,d\omega t\right]^{1/2} \] \[ V_{rms} = \left[\frac{V_m^2}{2\pi}\left(\frac{\omega t}{2} - \frac{\sin 2\omega t}{2}\right)_0^{\pi}\right]^{1/2} = \frac{V_m}{2} \] \[ I_{rms} = \frac{V_{rms}}{R_L} = \frac{V_m}{2R_L} = \frac{I_m}{2} \]

Average & RMS Voltage/Current for HWR – Non-Ideal Diode

\[ V_{dc} = \frac{V_m - V_k}{\pi} \] \[ I_{dc} = \frac{V_{dc}}{R_L} = \frac{V_m - V_k}{\pi R_L} = \frac{I_m}{\pi} \quad \text{where } I_m = \frac{V_m - V_k}{R_L} \] \[ V_{rms} = \frac{V_m - V_k}{2} \] \[ I_{rms} = \frac{V_{rms}}{R_L} = \frac{V_m - V_k}{2R_L} = \frac{I_m}{2} \quad \text{where } I_m = \frac{V_m - V_k}{R_L} \]

Ripple Factor of HWR

Ripple Factor is a measure of purity of the dc output of a rectifier and can be defined as:

\[ r = \frac{\text{rms value of the ac component of the output}}{\text{average or dc value of the output}} \] \[ \text{Ripple Factor } (\gamma_{HWR}) = \frac{V'_{rms}}{V_{dc}} = \frac{\sqrt{V_{rms}^2 - V_{dc}^2}}{V_{dc}} = \sqrt{\left(\frac{V_{rms}}{V_{dc}}\right)^2 - 1} = \sqrt{\left(\frac{V_m/2}{V_m/\pi}\right)^2 - 1} \] \[ V'_{rms} = \left[V_{rms}^2 - V_{dc}^2\right]^{1/2} = \left[\left(\frac{V_m}{2}\right)^2 - \left(\frac{V_m}{\pi}\right)^2\right]^{1/2} = 0.385\,V_m \] \[ \therefore \text{Ripple factor} = 1.21 \]

Half Wave Rectifier PIV – Ideal & Non-Ideal Diode

Peak Inverse Voltage (PIV) or Peak Reverse Voltage (PRV) — (For both Ideal & Non-Ideal) is the maximum reverse biased voltage (\(V_m\)) the diode can withstand without entering the breakdown region.

During the negative half cycle, \(I = 0\), so \(V_o = IR = (0)R = 0\,\text{V}\). The full source voltage appears across the reverse-biased diode.

\[ \boxed{\text{PIV rating} \geq V_m \quad \text{(half-wave rectifier)}} \]

Numericals on HWR

Problem 1a

Assuming an ideal diode, sketch \(V_i\), \(V_d\), and \(i_d\) for the half-wave rectifier. The input is a sinusoidal waveform with a frequency of 60 Hz. The circuit has a load resistor of 2.2 kΩ and \(V_{dc} = 2\,\text{V}\).

Solution

Given \(V_{dc} = 0.318\,V_m\):

\[ V_m = \frac{V_{dc}}{0.318} = \frac{2\,\text{V}}{0.318} = 6.28\,\text{V} \] \[ I_m = \frac{V_m}{R} = \frac{6.28\,\text{V}}{2.2\,\text{k}\Omega} = 2.85\,\text{mA} \]

Waveform sketches:

  • \(v_i\): Sinusoidal, peak = 6.28 V, trough = −6.28 V.
  • \(v_d\): During positive half: \(V_d = 0\,\text{V}\). During negative half: \(V_d = V_i\) (diode is reverse biased, so \(V_d\) swings to −6.28 V).
  • \(i_d\): Half-wave, peak = 2.85 mA, zero during negative half.
Problem 1b

Assuming a silicon diode (\(V_k = 0.7\,\text{V}\)), sketch \(v_i\), \(v_d\), and \(i_d\) for the half-wave rectifier. The input is a sinusoidal waveform with a frequency of 60 Hz. \(V_{dc} = 2\,\text{V}\), load resistor = 2.2 kΩ.

Solution

Using \(V_{dc} \cong 0.318(V_m - V_k)\):

\[ 2\,\text{V} = 0.318(V_m - 0.7\,\text{V}) \] \[ \text{Solving: } V_m = 6.98\,\text{V} \]

Output peak: \(V_{R,\,peak} = V_m - V_k = 6.98 - 0.7 = 6.28\,\text{V}\)

\[ I_m = \frac{6.28\,\text{V}}{2.2\,\text{k}\Omega} = 2.85\,\text{mA} \]
  • \(v_i\): Sinusoidal, peak = 6.98 V.
  • \(v_d\): Positive half: flat at 0.7 V during conduction. Negative half: goes to −6.98 V.
  • \(v_R\): Half-wave, peak = 6.28 V.
  • \(i_R (= i_d)\): Half-wave, peak = 2.85 mA.
Problem 2

(a) Given \(P_{max} = 14\,\text{mW}\) for each diode shown (silicon), determine the maximum current rating of each diode (using the approximate equivalent model).
(b) Determine \(I_{max}\) for \(V_{i\,max} = 160\,\text{V}\). (Two diodes in parallel: 4.7 kΩ ∥ 56 kΩ load.)
(c) Determine the current through each diode at \(V_{i\,max}\) using the results of part (b).
(d) If only one diode were present, determine the diode current and compare it to the maximum rating.

Solution

(a)

\[ P_{max} = 14\,\text{mW} = (0.7\,\text{V})\,I_D \implies I_D = \frac{14\,\text{mW}}{0.7\,\text{V}} = 20\,\text{mA} \]

(b)

\[ 4.7\,\text{k}\Omega \,\|\, 56\,\text{k}\Omega = 4.34\,\text{k}\Omega \] \[ V_R = 160\,\text{V} - 0.7\,\text{V} = 159.3\,\text{V} \] \[ I_{max} = \frac{159.3\,\text{V}}{4.34\,\text{k}\Omega} = 36.71\,\text{mA} \]

(c)

\[ I_{diode} = \frac{I_{max}}{2} = \frac{36.71\,\text{mA}}{2} = 18.36\,\text{mA} \quad (I_D = 20\,\text{mA} > 18.36\,\text{mA}) \]

(d)

\[ I_{diode} = 36.71\,\text{mA} \gg I_{max} = 20\,\text{mA} \quad \textbf{Total damage} \]

Full Wave Rectifier (FWR)

Full Wave Rectifier: Centre Tap (Ideal Diode)

A centre-tap transformer with turns ratio 1:2 provides two equal secondary voltages \(V_{AC}\) and \(V_{CB}\), each equal to \(V_i\), with \(V_{AB} = 2V_i\). Two diodes \(D_1\) and \(D_2\) conduct on alternating half-cycles.

Circuit Description: 1:2 centre-tap transformer. Terminal A connects through \(D_1\), terminal B through \(D_2\), centre tap (CT) is the ground reference. Load R is connected from the diode junction to CT.
Waveforms: \(V_{AC}\) or \(V_{CB}\) is a standard sine wave (peak \(V_m\)). \(V_{AB} = 2V_i\) has peak \(2V_m\).

Positive Half Cycle:

Diode \(D_1\) conducts; \(D_2\) is reverse-biased.

\[ V_o = V_m, \quad I_o = \frac{V_m}{R_L} \]

Negative Half Cycle:

Diode \(D_2\) conducts; \(D_1\) is reverse-biased.

\[ V_o = V_m, \quad I_m = \frac{V_m}{R_L} \]

Both half-cycles produce a positive output, giving a full-wave rectified signal with \(V_{dc} = 0.636\,V_m\).

Full Wave Rectifier PIV – Centre Tap (Ideal Diode)

When one diode is conducting, the other is reverse-biased. The voltage across the non-conducting diode equals the secondary voltage plus the output voltage.

\[ \text{PIV} = V_{secondary} + V_R = V_m + V_m \] \[ \boxed{\text{PIV} \geq 2V_m \quad \text{(CT transformer, full-wave rectifier)}} \]

Full Wave Rectifier: Centre Tap (Non-Ideal Diode)

Negative Half Cycle:

\[ V_{o\,max} = V_m - V_k, \quad I_{o\,max} = \frac{V_m - V_k}{R_L} \]

PIV for Non-Ideal Diode:

\[ \text{PIV} - V_m + V_k - V_m = 0 \implies \text{PIV} - 2V_m + V_k = 0 \implies \text{PIV} = 2V_m - V_k \] \[ \text{Non-Ideal diode: } \text{PIV} \geq 2V_m - V_k \]

Full Wave Rectifier: Bridge Rectifier (Ideal Diode)

Four diodes (\(D_1\), \(D_2\), \(D_3\), \(D_4\)) arranged in a bridge configuration. Load R is connected across the central nodes.

Bridge Rectifier Circuit: Four diodes in a diamond arrangement. \(D_1\) and \(D_4\) conduct on positive half-cycle; \(D_2\) and \(D_3\) conduct on negative half-cycle. Current always flows in the same direction through the load R.

Positive Half Cycle:

\(D_3\) "on", \(D_4\) "on"; \(D_1\) "off", \(D_2\) "off".

\[ V_o = V_m, \quad I_o = \frac{V_m}{R_L} \]

Negative Half Cycle:

\(D_1\) "on", \(D_2\) "on"; \(D_3\) "off", \(D_4\) "off".

\[ V_o = V_m, \quad I_o = \frac{V_m}{R_L} \]
Input Waveform: Sinusoidal, peak \(V_m\), trough \(-V_m\).
Output Waveform: Full-wave rectified, all positive, peak \(V_m\), \(V_{dc} = 0.636\,V_m\).

Full Wave Rectifier: Bridge Rectifier (Non-Ideal Diode)

With two diodes conducting per half-cycle, each dropping \(V_T = 0.7\,\text{V}\):

\[ v_i - V_T - v_o - V_T = 0 \implies v_o = v_i - 2V_T \implies V_{o\,max} = V_m - 2V_T \] \[ \boxed{V_{dc} \cong 0.636(V_m - 2V_T)} \]

Full Wave Rectifier PIV: Bridge Rectifier (Ideal & Non-Ideal)

Ideal Diode:

\[ \text{PIV} - V_m = 0 \implies \text{PIV} = V_m \] \[ \boxed{\text{PIV} \geq V_m \quad \text{(full-wave bridge rectifier)}} \]

Non-Ideal Diode:

\[ \text{PIV} - V_k - V_o = 0 \implies \text{PIV} - V_k - V_m + 2V_k = 0 \implies \text{PIV} + V_k - V_m = 0 \implies \text{PIV} = V_m - V_k \] \[ \text{Non-Ideal diode: } \text{PIV} \geq V_m - V_k \]

Determining Average Voltage/Current for FWR (Ideal Diode)

\[ V_{dc} = \frac{1}{\pi}\int_0^{\pi} V_m \sin\omega t\, d\omega t = \frac{V_m}{\pi}\left[-\cos\omega t\right]_0^{\pi} = \frac{2V_m}{\pi} \] \[ I_{dc} = \frac{V_{dc}}{R} = \frac{2V_m}{\pi R} = \frac{2I_m}{\pi} \]

Determining RMS Voltage/Current for FWR (Ideal Diode)

\[ V_{rms} = \left[\frac{1}{\pi}\int_0^{\pi}(V_m\sin\omega t)^2\,d\omega t\right]^{1/2} = \left[\frac{V_m^2}{\pi}\int_0^{\pi}\frac{1-\cos 2\omega t}{2}\,d\omega t\right]^{1/2} \] \[ = \left[\frac{V_m^2}{2\pi}\int_0^{\pi}(1-\cos 2\omega t)\,d\omega t\right]^{1/2} = \left[\frac{V_m^2}{2\pi}\left(\omega t - \frac{\sin 2\omega t}{2}\right)_0^{\pi}\right]^{1/2} = \left[\frac{V_m^2}{2\pi}(\pi - 0)\right]^{1/2} = \frac{V_m}{\sqrt{2}} \] \[ I_{rms} = \frac{V_{rms}}{R} = \frac{V_m}{\sqrt{2}\,R} = \frac{I_m}{\sqrt{2}} \]

Average & RMS Voltage/Current for Centre Tap Rectifier – Non-Ideal Diode

\[ V_{dc} = \frac{2(V_m - V_k)}{\pi} \] \[ I_{dc} = \frac{V_{dc}}{R_L} = \frac{2(V_m - V_k)}{\pi R_L} = 2\frac{I_m}{\pi} \quad \left(I_m = \frac{V_m - V_k}{R_L}\right) \] \[ V_{rms} = \frac{V_m - V_k}{\sqrt{2}} \] \[ I_{rms} = \frac{V_{rms}}{R_L} = \frac{V_m - V_k}{\sqrt{2}\,R_L} = \frac{I_m}{\sqrt{2}} \quad \left(I_m = \frac{V_m - V_k}{R_L}\right) \]

Average & RMS Voltage/Current for Bridge Rectifier – Non-Ideal Diode

\[ V_{dc} = \frac{2(V_m - 2V_k)}{\pi} \] \[ I_{dc} = \frac{V_{dc}}{R_L} = \frac{2(V_m - 2V_k)}{\pi R_L} = 2\frac{I_m}{\pi} \quad \left(I_m = \frac{V_m - 2V_k}{R_L}\right) \] \[ V_{rms} = \frac{V_m - 2V_k}{\sqrt{2}} \] \[ I_{rms} = \frac{V_{rms}}{R_L} = \frac{V_m - 2V_k}{\sqrt{2}\,R_L} = \frac{I_m}{\sqrt{2}} \quad \left(I_m = \frac{V_m - 2V_k}{R_L}\right) \]

Ripple Factor of FWR

Ripple Factor is a measure of purity of the dc output of a rectifier and can be defined as:

\[ r = \frac{\text{rms value of the ac component of the output}}{\text{average or dc value of the output}} \] \[ \text{Ripple Factor } (\gamma_{FWR}) = \frac{V'_{rms}}{V_{dc}} = \sqrt{\left(\frac{V_{rms}}{V_{dc}}\right)^2 - 1} = \sqrt{\left(\frac{V_m/\sqrt{2}}{2V_m/\pi}\right)^2 - 1} \] \[ V'^I_{rms} = \sqrt{V_{rms}^2 - V_{dc}^2} = \sqrt{\left(\frac{V_m}{\sqrt{2}}\right)^2 - \left(\frac{2V_m}{\pi}\right)^2} = 0.307\,V_m \] \[ \therefore \text{Ripple factor} = 0.48 \]

Numericals on Full Wave Rectifier: Bridge Rectifier

Problem 1

Determine the output waveform for the network and calculate the output dc level. Given: \(v_i\) peak = 10 V. Bridge rectifier with \(R_1 = 2\,\text{k}\Omega\), \(R_2 = 2\,\text{k}\Omega\), \(R_3 = 2\,\text{k}\Omega\). Output \(v_o\) is across \(R_3\).

Solution

Using Voltage Divider Rule:

\[ V_{o\,max} = \frac{2}{4}\,V_{i\,max} = \frac{2}{4}(10\,\text{V}) = 5\,\text{V} \] \[ V_0 = \frac{V_i R_1}{R_1 + R_2} = \frac{10(2\,\text{k})}{2\,\text{k} + 2\,\text{k}} = 5\,\text{V} \]

The output is full-wave rectified with peak = 5 V.

\[ V_{dc} = 0.636(5\,\text{V}) = 3.18\,\text{V} \]
Problem 2 — Sketch \(v_o\)

Sketch \(v_o\) for the network with ideal diodes, \(v_i\) peak = 170 V, \(-170\,\text{V}\), load resistors = 2.2 kΩ each in the bridge arms.

Solution

Positive pulse of \(v_i\): Top left diode "off", bottom left diode "on".

\[ 2.2\,\text{k}\Omega \,\|\, 2.2\,\text{k}\Omega = 1.1\,\text{k}\Omega \] \[ V_{o_{peak}} = \frac{1.1\,\text{k}\Omega(170\,\text{V})}{1.1\,\text{k}\Omega + 2.2\,\text{k}\Omega} = 56.67\,\text{V} \]

Negative pulse of \(v_i\): Top left diode "on", bottom left diode "off".

\[ V_{o_{peak}} = \frac{1.1\,\text{k}\Omega(170\,\text{V})}{1.1\,\text{k}\Omega + 2.2\,\text{k}\Omega} = 56.67\,\text{V} \] \[ V_{dc} = 0.636(56.67\,\text{V}) = 36.04\,\text{V} \]
Problem 3

Determine \(V_o\) and the required PIV rating of each diode for the configuration. In addition, determine the maximum current through each diode. Given: \(v_i\) peak = 100 V, \(R_L = 2.2\,\text{k}\Omega\), ideal diodes in special bridge orientation.

Solution

For positive cycle — Path: ABDCEF

Applying KVL: \(V_i + V_0 = 0 \implies V_0 = -V_i\)

For negative cycle — Path: FEDCBA

Output waveform: inverted full-wave, peak = −100 V (negative-going full wave).

\[ \text{PIV} = 100\,\text{V} \] \[ I_{max} = \frac{100\,\text{V}}{2.2\,\text{k}\Omega} = 45.45\,\text{mA} \]
Problem 4

Sketch \(V_o\) for the network and determine the dc voltage available. Given: \(v_i\) peak = 100 V, ideal diodes, bridge with \(R_2 = R_3 = 2.2\,\text{k}\Omega\) in the bridge, \(R_{load} = 2.2\,\text{k}\Omega\).

Solution

Positive half-cycle of \(v_i\) — Network redrawn:

Voltage-divider rule:

\[ V_{o_{max}} = \frac{2.2\,\text{k}\Omega(V_{i_{max}})}{2.2\,\text{k}\Omega + 2.2\,\text{k}\Omega} = \frac{1}{2}(V_{i_{max}}) = \frac{1}{2}(100\,\text{V}) = 50\,\text{V} \]

Negative half-cycle of \(v_i\):

\[ V_{o_{max}} = \frac{2.2\,\text{k}\Omega(V_{i_{max}})}{2.2\,\text{k}\Omega + 2.2\,\text{k}\Omega} = \frac{1}{2}(100\,\text{V}) = 50\,\text{V} \] \[ V_{dc} = 0.636\,V_m = 0.636(50\,\text{V}) = 31.8\,\text{V} \]
Problem 5

A full-wave bridge rectifier with a 120-V rms sinusoidal input has a load resistor of 1 kΩ.
(a) If silicon diodes are employed, what is the dc voltage available at the load?
(b) Determine the required PIV rating of each diode.
(c) Find the maximum current through each diode during conduction.
(d) What is the required power rating of each diode?

Solution

(a)

\[ V_m = \sqrt{2}(120\,\text{V}) = 169.7\,\text{V} \] \[ V_{L_m} = V_{i_m} - 2V_D = 169.7\,\text{V} - 2(0.7\,\text{V}) = 169.7 - 1.4 = 168.3\,\text{V} \] \[ V_{dc} = 0.636(168.3\,\text{V}) = 107.04\,\text{V} \]

(b)

\[ \text{PIV}: V_m - V_k = 169.7 - 0.7 = 169\,\text{V} \]

(c)

\[ I_D(\max) = \frac{V_{L_m}}{R_L} = \frac{168.3\,\text{V}}{1\,\text{k}\Omega} = 168.3\,\text{mA} \]

(d)

\[ P_{max} = V_D I_D = (0.7\,\text{V})(168.3\,\text{mA}) = 117.81\,\text{mW} \]
Problem 6

The input to a bridge rectifier is given through a 10:1 transformer from a supply of 230 Sin 314t. If \(R_L = 500\,\Omega\). Determine: DC load voltage, RMS load voltage, PIV across diodes, DC power delivered to the load.

Solution

The primary voltage of transformer, \(V_1 = 230\,\sin 314t\).

Transformer turns ratio: \(\dfrac{N_1}{N_2} = \dfrac{10}{1}\)

The secondary voltage of transformer:

\[ V_2 = \frac{N_2}{N_1}\,V_1 = \frac{1}{10}(230\,\sin 314t) = 23\,\sin 314t \] \[ V_2 = V_m\,\sin\omega t = 23\,\sin 314t \implies V_m = 23\,\text{V} \] \[ \text{DC load voltage:}\quad V_{dc} = \frac{2V_m}{\pi} = \frac{2 \times 23}{\pi} = 14.64\,\text{V} \] \[ \text{RMS load voltage:}\quad V_{rms} = \frac{V_m}{\sqrt{2}} = \frac{23}{\sqrt{2}} = 16.26\,\text{V} \] \[ \text{PIV across diodes: PIV} \geq V_m \geq 23\,\text{V} \] \[ \text{DC power delivered to load:}\quad P_{dc} = I_{dc}^2 \times R_L = \frac{V_{dc}^2}{R_L} = \frac{(14.64)^2}{500} = 0.428\,\text{W} \]

Shunt Capacitor Filter

Filter is a circuit which converts pulsating DC to smooth DC. Since the reactance of capacitor and inductor depends on frequency, by connecting a capacitor in parallel with the load or by connecting an inductor in series with the load, the ac ripple present at the output of a rectifier can be reduced.

Basic Capacitor Filter (Fig 15.3):
AC input → Rectifier circuit → [Capacitor C in parallel with DC load \(R_L\)] → Output
The capacitor filters the ripple from the rectifier output, providing a smoother DC voltage to the load.

FWR with Shunt Capacitor Filter

FWR with Shunt Capacitor Filter Circuit:
230 V, 50 Hz AC Supply → Centre-tap transformer → \(D_1\) (from terminal A) and \(D_2\) (from terminal B) → Junction point X → Capacitor C in parallel with \(R_L\) → Output \(V_o\) → Return to centre tap Y.
Input: \(V_i = V_m \sin\omega t\) for each secondary half.

Working

During positive half cycle the diode D1 is forward biased in centre tap full wave rectifier and diode D1 and D2 are forward biased in bridge rectifier. Hence the capacitor C will charge from 0 V towards the peak value of input \(V_m\). When input voltage \(V_i\) reaches \(V_m\) the charge on capacitor will also be equal to \(V_m\), hence the diodes will turn off (since both anode and cathode of diodes will be at same potential). Now the capacitor starts discharging through the load \(R_L\), if the load is very light (load resistance is very high and load current is low) the capacitor will discharge by a small amount, by that time next half cycle charges the capacitor back to \(V_m\). Thus the output voltage remains almost constant.

Capacitor Filter — Output Voltage Waveform

Output voltage waveform with capacitor filter:
The waveform shows the capacitor charging up to \(V_m\) during time \(T_1\) (diodes conducting), then discharging slightly during time \(T_2\) (diodes off). Peak-to-peak ripple voltage \((V_r)_{p-p}\) is marked. Average DC level \(V_{dc}\) is close to \(V_m\). Period of the rectified waveform is \(T/2\) for a full-wave rectifier.
  • Time T1: The time during which diodes of the full-wave rectifier conduct, charging the capacitor up to the peak rectifier voltage \(V_m\).
  • Time T2: The time interval during which the rectifier voltage drops below the peak voltage, and the capacitor discharges through the load.
  • Since the charge–discharge cycle occurs for each half-cycle for a full-wave rectifier, the period of the rectified waveform is \(T/2\).

Expression for Ripple Factor — Shunt Capacitor Filter (FWR)

\[ \gamma = \frac{1}{4\sqrt{3}\,f\,C\,R_L} \] Where: \begin{align} f &\to \text{Frequency of input AC signal}\\ C &\to \text{Filter capacitor used}\\ R_L &\to \text{Load resistance} \end{align} \[ \gamma \propto \frac{1}{f} \cdot \frac{1}{C} \cdot \frac{1}{R_L} \]

Since ripple is inversely proportional to frequency, capacitor and load resistance, the ripple can be minimized by using high value of C. (\(R_L\) and \(f\) are not in designer's hand.)

\[ \gamma = \frac{V_{r(rms)}}{V_{dc}}, \quad V_{r(rms)} = \frac{I_{DC}}{4\sqrt{3}\,fc} \] \[ V_{dc} = V_m - \frac{I_{DC}}{4\,fc} \]
Shunt Capacitor Filter for Bridge Rectifier Circuit:
230 V, 50 Hz AC → Transformer → Bridge (D4, D1, D3, D2) → X → Capacitor C in parallel with \(R_L\) → Output \(V_o\) → Y.
Waveform: Without capacitor — pulsating full-wave. With capacitor — ripple is greatly reduced, output stays close to \(V_m\) with small ripple (C charges / C discharges shown on waveform).
  • AC components fluctuate with respect to time while the DC components remain constant with respect to time.
  • So the AC components present in the pulsating DC is an unwanted signal.

Shunt Capacitor Filter for Half-Wave Rectifier

Capacitor C is connected in shunt/parallel with load resistor \(R_L\).

HWR with C Filter Circuit:
AC Input → Transformer → Diode D → Resistor R (series) → Smoothing Capacitor C (in parallel) → Output.
AC Input Waveform: Sinusoidal, ±\(V_{max}\).
Resultant Output Waveform: C charges on positive peaks, C discharges between peaks, output is smoother than without capacitor.
Path of components: AC component (high frequency) flows through C; DC component (low frequency) flows through \(R_L\).

Ripple Factor for Capacitor Filter of a Half Wave Rectifier is given by:

\[ \text{Ripple Factor for a capacitor filter (HWR)} = \frac{1}{2\sqrt{3}\,f\,C\,R_L} \] \[ f\text{ – Frequency of AC input Signal}, \quad R_L\text{ – Value of Load resistor}, \quad C\text{ – Capacitance of the Shunt Capacitor} \] \[ \gamma = \frac{V_{r(rms)}}{V_{dc}}, \quad V_{r(rms)} = \frac{I_{DC}}{2\sqrt{3}\,fc} \] \[ V_{dc} = V_m - \frac{I_{DC}}{2\,fc} \]

Unit II: Zener Diode

  • Heavily Doped PN-Junction Device Specially Designed to operate under breakdown region.
  • Zener Diode under forward Bias condition works like a normal semiconductor diode.
  • Zener Diode under reverse bias condition acts like Voltage regulator.

Zener diodes are usually heavily doped diodes hence the depletion layer is very narrow. When the reverse voltage across the junction is increased the electric field across the depletion layer becomes high so that electrons are pulled out of covalent bonds resulting in a sudden rise in current which is known as zener breakdown.

Zener Diode Characteristics

Structure of Zener Diode: P-type | N-type with depletion region. Heavily doped (\(p^+\) | \(n^+\)). Negative ions on P-side, positive ions on N-side in depletion region.

Symbol of Zener Diode: Like a regular diode but with bent ends on the cathode bar (the black colour ring denotes the negative terminal of the Zener diode). Circuit symbol: P→N with bent cathode.

V-I Characteristics of Zener Diode:
— In Forward Bias Condition: Behaves like a normal diode; current increases rapidly above cut-in voltage (~0.7 V for Si).
— In Reverse Bias Condition: Small reverse current (\(I_o\)) until breakdown at \(-V_Z\); then current rises sharply (zener breakdown region). The voltage remains nearly constant at \(V_Z\) for a wide range of currents from \(I_{Z(min)}\) to \(I_{Z(max)}\). This is the "Constant Zener Voltage" region.
Graph showing equivalent circuit in each of the regions:
— Forward bias above 0.7 V: Equivalent to a 0.7 V voltage source with small series resistance \(r_Z\).
— Reverse bias before breakdown: Equivalent to open circuit (very small leakage \(I_R = 10\,\mu\text{A}\)).
— Breakdown region (at \(V_Z\)): Equivalent to a \(V_Z\) voltage source with small dynamic resistance \(r_Z = \Delta V_Z / \Delta I_Z\). At test current \(I_{ZT} = 12.5\,\text{mA}\), \(r_Z = 8.5\,\Omega\). Maximum current \(I_{ZM} = 32\,\text{mA}\).

Zener Diode as Voltage Regulator

Conditions required to operate Zener Diode as Voltage Regulator:

  • Zener Diode should be in Reverse Biased Condition.
  • \(V_{in}\) should be Greater than \(V_Z\).
  • \(I_Z\) should be greater than \(I_{Zmin}\).
  • \(I_Z\) should be less than or equal to \(I_{Zmax}\).
Zener Voltage Regulator Circuit:
\(V_i\) (variable input) → Series resistor R → Node → Zener diode \(V_Z\) (in parallel) → Ground.
Load \(R_L\) in parallel with Zener. \(I_Z\) flows through Zener, \(I_L\) flows through \(R_L\), \(I_R = I_Z + I_L\) flows through R.

Different operating conditions of Zener Regulator:

  1. Fixed quantities (\(V_i\) and R are fixed)
  2. Fixed supply voltage and a variable load (Fixed \(V_i\) and variable \(R_L\))
  3. Fixed load and a variable supply (Fixed \(R_L\) and Variable \(V_i\))
  4. Variable Supply and Variable Load (Variable \(V_i\) and Variable \(R_L\))

CASE-1: Fixed \(V_i\) and Fixed \(R_L\) Voltage Regulator

The analysis can fundamentally be broken down into two steps.

1. Determine the state of the Zener diode by removing it from the network and calculating the voltage across the resulting open circuit:

\[ V = V_L = \frac{R_L\,V_i}{R + R_L} \]
  • If \(V \geq V_Z\), the Zener diode is on, and the appropriate equivalent model can be substituted.
  • If \(V < V_Z\), the diode is off, and the open-circuit equivalence is substituted.

Conclusion:

If the Zener diode is in the "on" state, the voltage across the diode is not V volts. The Zener diode will turn on as soon as the voltage across the Zener diode is \(V_Z\) volts. It will then "lock in" at this level and never reach the higher level of V volts.

Problem 1 — Zener Regulator

For the Zener diode network: \(V_i = 16\,\text{V}\), \(R = 1\,\text{k}\Omega\), \(V_Z = 10\,\text{V}\), \(P_{ZM} = 30\,\text{mW}\).
(i) Determine \(V_L\), \(V_R\), \(I_Z\), and \(P_Z\) with \(R_L = 1.2\,\text{k}\Omega\).
(ii) Repeat part (a) with \(R_L = 3\,\text{k}\Omega\).

Solution (i)
\[ V = \frac{R_L V_i}{R + R_L} = \frac{1.2\,\text{k}\Omega(16\,\text{V})}{1\,\text{k}\Omega + 1.2\,\text{k}\Omega} = 8.73\,\text{V} \]

Since \(V = 8.73\,\text{V} < V_Z = 10\,\text{V}\), the Zener diode is OFF.

\[ V_L = V = 8.73\,\text{V} \] \[ V_R = V_i - V_L = 16\,\text{V} - 8.73\,\text{V} = 7.27\,\text{V} \] \[ I_Z = 0\,\text{A}, \quad P_Z = V_Z I_Z = V_Z(0\,\text{A}) = 0\,\text{W} \]
Solution (ii)
\[ V = \frac{R_L V_i}{R + R_L} = \frac{3\,\text{k}\Omega(16\,\text{V})}{1\,\text{k}\Omega + 3\,\text{k}\Omega} = 12\,\text{V} \]

Since \(V = 12\,\text{V} > V_Z = 10\,\text{V}\), the Zener diode is ON.

\[ V_L = V_Z = 10\,\text{V}, \quad V_R = V_i - V_L = 16 - 10 = 6\,\text{V} \] \[ I_L = \frac{V_L}{R_L} = \frac{10\,\text{V}}{3\,\text{k}\Omega} = 3.33\,\text{mA} \] \[ I_R = \frac{V_R}{R} = \frac{6\,\text{V}}{1\,\text{k}\Omega} = 6\,\text{mA} \] \[ I_Z = I_R - I_L = 6\,\text{mA} - 3.33\,\text{mA} = 2.67\,\text{mA} \]

The power dissipated is:

\[ P_Z = V_Z I_Z = (10\,\text{V})(2.67\,\text{mA}) = 26.7\,\text{mW} \]

which is less than the specified \(P_{ZM} = 30\,\text{mW}\). ✓

Problem 2 — Zener Regulator

Given: 20 V supply, \(R_S = 220\,\Omega\), \(V_Z = 10\,\text{V}\), \(P_{Z_{max}} = 400\,\text{mW}\).
(a) Determine \(V_L\), \(I_L\), \(I_Z\), and \(I_R\) if \(R_L = 180\,\Omega\).
(b) Repeat part (a) if \(R_L = 470\,\Omega\).

Solution (a)

In the absence of the Zener diode:

\[ V_L = \frac{180\,\Omega(20\,\text{V})}{180\,\Omega + 220\,\Omega} = 9\,\text{V} \]

\(V_L = 9\,\text{V} < V_Z = 10\,\text{V}\) and diode non-conducting.

\[ \text{Therefore, } I_L = I_R = \frac{20\,\text{V}}{220\,\Omega + 180\,\Omega} = 50\,\text{mA} \]

with \(I_Z = 0\,\text{mA}\) and \(V_L = 9\,\text{V}\).

Solution (b)

In the absence of the Zener diode:

\[ V_L = \frac{470\,\Omega(20\,\text{V})}{470\,\Omega + 220\,\Omega} = 13.62\,\text{V} \]

\(V_L = 13.62\,\text{V} > V_Z = 10\,\text{V}\) and Zener diode "on".

\[ \text{Therefore, } V_L = 10\,\text{V} \text{ and } V_{R_S} = 10\,\text{V} \] \[ I_{R_S} = V_{R_S}/R_S = 10\,\text{V}/220\,\Omega = 45.45\,\text{mA} \] \[ I_L = V_L/R_L = 10\,\text{V}/470\,\Omega = 21.28\,\text{mA} \] \[ I_Z = I_{R_S} - I_L = 45.45\,\text{mA} - 21.28\,\text{mA} = 24.17\,\text{mA} \]

CASE-2: Fixed Supply Voltage and a Variable Load

Conditions for \(R_L\) to keep the Zener current \(I_Z\) in the operating limit: \(I_{Zmin} \leq I_Z < I_{Zmax}\).

Condition for \(R_{L\,min}\):

\[ I_L = I_R - I_Z, \quad R_L = \frac{V_Z}{I_L} \quad (R_L \text{ is Minimum when } I_L \text{ is Maximum}) \] \[ I_{Lmax} = I_R - I_{Zmin} \quad \text{where } I_R = \frac{V_i - V_Z}{R} \] \[ R_{Lmin} = \frac{V_Z}{I_{Lmax}} \] \text{If } I_{Zmin} = 0 \text{ then } I_{Lmax} = I_R \text{, hence: } \[ R_{Lmin} = \frac{V_Z}{I_R} = \frac{V_Z\,R}{V_i - V_Z} \]

Condition for \(R_{L\,max}\):

\[ R_L \text{ is Maximum when } I_L \text{ is Minimum} \] \[ I_{Lmin} = I_R - I_{Zmax} \quad \text{where } I_R = \frac{V_i - V_Z}{R} \text{ and } I_{Zmax} = \frac{P_{ZM}}{V_Z} \] \[ R_{Lmax} = \frac{V_Z}{I_{Lmin}} \]

For the network: \(V_i = 50\,\text{V}\), \(R = 1\,\text{k}\Omega\), \(V_Z = 10\,\text{V}\), \(I_{ZM} = 32\,\text{mA}\).

CASE-4: Variable Supply Voltage and a Variable Load

Worst case conditions for Zener Diode:

1. For the diode to enter into the breakdown region, the necessary condition is \(I_Z \geq I_{Zmin}\).

\(I_Z\) can become less than \(I_{Zmin}\) when:

  1. \(I_L\) increases or \(R_L\) decreases.
  2. R increases or \(I_R\) decreases.
  3. \(V_{in}\) decreases.

Hence the value for \(R_{max}\) to ensure \(I_Z \geq I_{Zmin}\) is:

\[ R_{max} = \frac{V_{imin} - V_Z}{I_{Zmin} + I_{Lmax}} \]

2. The upper limit for the Zener current is \(I_Z < I_{Zmax}\).

\(I_Z\) can become greater than \(I_{Zmax}\) when:

  1. \(I_L\) decreases or \(R_L\) increases.
  2. R decreases or \(I_R\) increases.
  3. \(V_{in}\) increases.

Hence the value for \(R_{min}\) to ensure \(I_Z < I_{Zmax}\) is:

\[ R_{min} = \frac{V_{imax} - V_Z}{I_{Zmax} + I_{Lmin}} \]
